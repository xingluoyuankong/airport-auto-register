## 八、Outlook邮箱验证码技能使用说明

**触发关键词**: outlook验证码、邮箱验证码、查收邮件、读取验证码、outlook code、邮件验证

**Python导入**:
```python
from outlook_skill import OutlookVerifier

ov = OutlookVerifier()
# IMAP模式（最稳定）
result = ov.get_code_imap("email@outlook.com", "password", timeout=60)
# 自动模式
result = ov.get_code_auto("email", "password", sender_filter="airport", timeout=120)
# 等验证链接
result = ov.wait_for_link("email", "password", sender_filter="noreply@v2ny.com")
# 返回: {"success": True, "code": "123456", "links": ["https://..."], ...}
```
