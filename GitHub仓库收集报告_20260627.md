## 二、现有工具盘点 (01-机场VPN注册机)

### 已部署的工具

| 工具 | 类型 | 支持的机场类型 | 状态 |
|------|------|---------------|------|
| **hidexx** | Go CLI | hidexx/aiguobit SSPanel 变种 | ✅ 可用 |
| **v2register** | Python | V2Board API 直接注册 (需无验证码) | ✅ 可用 |
| **zetsalvpn-fresh** | TypeScript/Puppeteer | ZetsaVPN 图形验证码识别 | ✅ 可用 |
| **zetsalvpn-gen** | TypeScript/Puppeteer | ZetsaVPN 账号生成(OCR) | ✅ 可用 |
| **auto_register.py** | Python/ddddocr | hidexx SSPanel 系统 (4位验证码) | ✅ 可用 |
| **multi_airport_register.py** | Python | 多机场批量注册 | ✅ 可用 |
| **Warp-auto-register** | Python | Cloudflare WARP (多线程) | ✅ 可用 |

### 实测结果 (2026-06-04)

- **hidexx 系统 9 个站点运行成功**: hidexx-a~c + 倚天剑 a~f
- **成功率**: 9/27 目标 (33%), 但多数失败是因为站点已失效
- **核心能力**: ddddocr 识别 4位字母数字验证码, >95%

## 四、支持 Outlook 邮箱注册的机场

基于 hidexx 系统的实测,以下类型天然支持任意邮箱(含 Outlook):

1. **hidexx 系列** (SSPanel 变种, 不验证邮箱)
2. **aiguobit/倚天剑系列** (同上)
3. **V2Board 未开验证的站点** (可通过 v2register 批量注册)
