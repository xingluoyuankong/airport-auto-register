## ✅ hidexx（Go）

- **功能**: Cloudflare WARP 账号注册 → OCR验证码 → 领取免费试用 → 生成Clash订阅
- **注册账号**: `hx9ruiifg5@outlook.com` / `Hx@f0J48ax4vd`
- **主订阅链接**: `https://sub.51trojan.com/v1/sub/63915/60ba67964091ddfc838ea554bec599e8.yaml`
- **注意**: 本地编译需 `go build`，首次运行会自动安装 tesseract

## ✅ v2register（Python）

- **功能**: V2Board 机场自动注册（API模式）
- **目标机场**: `https://fastestcloud.xyz/`
- **注册结果**: 20个账号，全部含 JWT token 和 auth_data
- **账号文件**: `v2register/accounts.txt`
- **修改**: 将目标URL从 `xxxxxx` 改为实际机场地址，线程数从50降到5（测试用）

## ❌ Warp-auto-register（Python+Selenium）

- **功能**: Cloudflare WARP 账号自动注册
- **卡点**: 依赖 `tempmail.lol` 获取临时邮箱，该API已不可用（返回错误）
- **解决方向**: 替换为其他临时邮箱服务（如 guerrillamail, tempmail.plus）

## ❌ zetsalvpn-gen（TypeScript/Python）

- **功能**: Zetsal VPN 自动注册+领取免费试用
- **卡点**: 注册页面有6位数字验证码，含大量灰度噪点
  - tesseract 识别准确率 ~10%（经常少读/错读数字）
  - ddddocr 识别准确率 ~5%（输出乱码或位数不对）
  - 预处理（二值化、去噪线）未能有效提升准确率
- **解决方向**: 接入付费验证码识别API（如 2captcha, anticaptcha），或训练专用OCR模型

## 关键经验

1. **代理问题**: 系统配置了 Clash 代理（socks5h://127.0.0.1:7897），Python requests 默认继承系统代理，需显式设置 `proxies={'http': None, 'https': None}` 或清除环境变量
2. **编码问题**: PowerShell GBK 编码与中文路径/emoji冲突，需设 `PYTHONIOENCODING=utf-8` 或用 `chcp 65001`
3. **ChromeDriver**: undetected-chromedriver 下载的版本可能与本地 Chrome 不匹配，需用 `webdriver_manager` 或手动指定缓存路径
4. **验证码**: 简单灰度噪点验证码对 tesseract/ddddocr 仍是难题，需要专业服务
