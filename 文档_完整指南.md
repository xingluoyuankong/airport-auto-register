## 项目概述

本项目包含两套机场自动注册工具：

| 工具 | 语言 | 目标面板 | 路径 |
|------|------|----------|------|
| **hidexx** | Go | hidexx 系统 (表单注册 + 验证码 OCR) | `01-机场VPN注册机\hidexx\` |
| **v2register** | Python | V2Board 系统 (JSON API 注册) | `01-机场VPN注册机\v2register\` |

## 现有工具

### hidexx (Go)

- **原理**: 模拟浏览器表单提交，使用 tesseract/ddddocr 识别验证码
- **注册接口**: `POST /users/register` (表单: email, pass1, pass2, checkcode)
- **验证码**: `GET /users/vcode` (PNG 图片)
- **试用领取**: `POST /orders/request_day_trial` (表单: sid, checksum, line_id, quantity)
- **订阅获取**: `GET /users/ucenter` → 解析 `copyText('URL')` 模式
- **配置**: `config.example.yaml` → `config.yaml`
- **当前目标**: `a.hidexx.com`

### v2register (Python)

- **原理**: 直接调用 V2Board API，无需浏览器
- **注册接口**: `POST /api/v1/passport/auth/register` (JSON: email, password, email_code, invite_code)
- **订阅获取**: `GET /api/v1/user/getSubscribe` (Header: Authorization: auth_data)
- **当前目标**: `fastestcloud.xyz`
- **多线程**: 默认 5 线程，可自定义

### auto_register.py (新增)

- **整合脚本**: 同时支持 hidexx 和 V2Board 两种模式
- **自动验证码**: 使用 ddddocr 自动识别
- **完整流程**: 注册 → 领试用 → 获取订阅 → 保存结果
- **路径**: `workspace\auto_register.py`

## 机场面板类型与 API 格式

### 1. hidexx 系统

```
注册: POST /users/register
  Content-Type: application/x-www-form-urlencoded
  Body: email=xxx&pass1=xxx&pass2=xxx&checkcode=xxx

验证码: GET /users/vcode
  Response: PNG image (4位字母数字)

登录: POST /users/login
  Content-Type: application/x-www-form-urlencoded
  Body: email=xxx&password=xxx

用户中心: GET /users/ucenter

试用领取: POST /orders/request_day_trial
  Content-Type: application/x-www-form-urlencoded
  Body: sid=xxx&checksum=xxx&line_id=1&quantity=1

订阅链接: 从 ucenter 页面解析 copyText('URL') 模式
```

### 2. V2Board 系统

```
注册: POST /api/v1/passport/auth/register
  Content-Type: application/json
  Body: {"email":"xxx","password":"xxx","email_code":"","invite_code":"","recaptcha_data":""}

订阅: GET /api/v1/user/getSubscribe
  Header: Authorization: auth_data
  Response: {"data":{"subscribe_url":"..."}}
```

### 3. SSPanel 系统

```
注册: POST /auth/register
  Content-Type: application/x-www-form-urlencoded
  Body: email=xxx&passwd=xxx&repasswd=xxx&code=xxx
  (部分站点也兼容 V2Board API 格式)
```

### 邮箱来源

路径: `C:\Users\XZXyuan\Downloads\批量注册邮箱\`

格式: `账号----密码----client_id----refresh_token`

### 令牌取件系统

路径: `E:\API获取工具\邮箱自动批量注册\token-mail-app\`

启动:
```bash
cd backend
pip install -r requirements.txt
python main.py
```
访问: http://localhost:18080

API:
- `POST /api/fetch_by_token` - 通过 token 获取邮件
- `POST /api/accounts/import` - 导入账号
- `POST /api/merge_files` - 合并文件

### 邮箱验证流程

1. 注册时使用 Outlook 邮箱
2. 等待验证邮件到达
3. 通过 token-mail-app 获取邮件
4. 提取验证码 (正则: `\d{4,8}`)
5. 提交验证码完成注册

## 注意事项

1. **站点时效性**: 免费机场站点经常失效，需要定期更新目标列表
2. **验证码识别**: ddddocr 比 tesseract 更适合中文环境的简单验证码
3. **网络环境**: 部分站点可能需要代理才能访问
4. **频率控制**: 注册间隔建议 1-3 秒，避免触发风控
5. **邮箱选择**: 建议使用 Outlook/Hotmail 邮箱，存活率较高
6. **订阅更新**: 免费试用通常 1 天过期，需要定期重新注册
