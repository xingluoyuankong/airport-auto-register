## Speedy (cloud.speedypro.xyz)

- URL: https://cloud.speedypro.xyz/#/register
- 面板: 简化版V2Board
- 页面结构:
  - 邮箱: 前缀输入框 + 域名下拉框(@gmail.com/@qq.com/@outlook.com等9种)
  - 邮箱验证码: 输入框 + "发送"按钮
  - 密码 + 确认密码
  - 邀请码
  - "注册"按钮

- 注册步骤:
  1. 填邮箱前缀 → 选域名(@outlook.com)
  2. 填密码 → 确认密码
  3. 点"发送" → 弹出 Google reCAPTCHA!
  4. reCAPTCHA要求选择包含"公交车"的图片
  5. 完成reCAPTCHA → 等待邮箱验证码
  6. 填验证码 → 点"注册"

- 障碍: Google reCAPTCHA图片验证（需人工操作或reCAPTCHA求解服务）

- 关键结论: Speedy需要reCAPTCHA，自动化困难

## 大象网络 (elephant223.com)

- URL: https://www.elephant223.com/app#/register
- 面板: XBoard (V2Board变种，路径在/app#/)
- 页面结构:
  - 邮箱: 完整输入框（域名选择器固定@qq.com）
  - 邮箱验证码: 输入框 + "发送"按钮
  - 密码 + 确认密码
  - 邀请码(选填)
  - "注册"按钮

- 注册步骤:
  1. 填邮箱(@outlook.com完整地址)
  2. 填密码 → 确认密码
  3. 点"发送" → API返回422失败
  4. 直接点"注册" → 无反应

- 障碍: 
  - 发送验证码API返回422(可能需reCAPTCHA或邮箱格式验证)
  - 域名选择器只显示@qq.com(无法选择outlook)
  - 填入outlook完整地址后仍422

- 关键结论: 大象网络注册API有额外的验证限制，需要进一步逆向
