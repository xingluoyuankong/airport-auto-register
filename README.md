## 本会话关键教训

### 1. 代理地址纠正
- **正确**: `http://127.0.0.1:7897` (HTTP代理，支持CONNECT隧道)
- **错误**: 不要用 `https://127.0.0.1:7897` 也不是SOCKS5

### 2. 只用Outlook邮箱
- 用户明确要求：只用@outlook.com注册
- 优先找支持完整邮箱输入或下拉框含@outlook.com选项的机场

### 3. V2Board邮箱后缀用select元素（cd520成功关键）
- 邮箱输入是"前缀文本框 + `<select>`后缀下拉框"结构
- 正确: `page.selectOption('select', 'outlook.com')`
- 错误: 点击文本"suffix click"失败

### 4. V24批量结果
- 20个机场19个GOTO_FAILED (ERR_CONNECTION_CLOSED/Timeout)
- 仅TANZCLOUD注册成功(跳转login, 3天8G)
- **根因**: 大部分anran-world仓库里的机场域名已经死了

### 5. fastestcloud历史账户
- 25个fastestcloud.xyz账户(含订阅链接)在 `v2register/subscriptions.txt`
- 可使用任意一个token导入Clash试用
