## 关键教训

### 核心瓶颈：无免费额度
- 所有机场注册极容易（SSPANEL无验证码直接过，V2Board/Mala-Pro需邮箱验证码但Graph API可解）
- 真问题：没有任何机场注册后自动给免费套餐
- GLaDOS是唯一确认给4天真免费试用的，但拒所有Microsoft邮箱（hotmail/outlook全拒）
- 雨燕云等V2Board也拒Microsoft邮箱

### 已验证的注册系统分类
1. **SSPANEL** (稳连云/大象/三毛/魔戒)：无验证码/需验证码，VUE_NAIVE_ACCESS_TOKEN存localStorage，订阅链接= {域名}/api/v1/client/subscribe?token={uuid}
2. **Mala-Pro/Vue SPA** (宝可梦)：需验证码，API域名和面板域名分离，subscribe_url存localStorage
3. **hidexx** (aiguobit)：SSO认证，需验证码，无订阅链接只有购买后给节点
4. **扬帆云V2Board** (1元/2元)：强反爬，register→login死循环
5. **网际快车SSPANEL** (三毛)：中转网关c.xn--66tw07h.com→内部面板

### 技术栈
- **浏览器**: playwright-cli v0.1.14 + Edge Chromium (能穿透GFW访问被墙站点！Python requests不行)
- **验证码**: Microsoft Graph API (14个token账户可用)
- **playwright-cli eval限制突破**: 逗号操作符串联表达式 `(a=1, b=2, a+b)`；不能用var/function/for
- **订阅链接通用提取公式**: 登录→eval取VUE_NAIVE_ACCESS_TOKEN→fetch /api/v1/user/info→取uuid→拼URL
- **测试脚本**: test/文件夹，主项目代码严格隔离

```bash
# 清浏览器
taskkill /f /im chromium.exe

# 打开机场
cd "E:\API获取工具"
playwright-cli open <url> --browser=msedge

# 查页面
playwright-cli snapshot
playwright-cli eval "document.body.innerText.substring(0,500)"

# 填表
playwright-cli fill <ref> "value"
playwright-cli click <ref>

# 复杂JS（逗号操作符）
playwright-cli eval "(window.__x=1, localStorage.setItem('k','v'), document.querySelector('button').click())"

# 登录后查token
playwright-cli eval "localStorage.getItem('VUE_NAIVE_ACCESS_TOKEN')"

# fetch API提取uuid
playwright-cli eval "fetch('/api/v1/user/info',{headers:{'Authorization':'Bearer TOKEN'}}).then(function(r){return r.text()}).then(function(t){window.__info=t})"
```
