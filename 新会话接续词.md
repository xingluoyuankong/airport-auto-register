### 订阅链接获取方法（核心教训！）
**绝不能用UUID拼链接！** 每个机场的真实订阅域名和面板域名完全不同：
- FLYBIT面板: flybit.vip → 真实订阅: s.fb.22na.cn
- 99吧面板: a.99ba2026.fyi → 真实订阅: sealosgzg.site

- TaiShan面板: taishan.pro → 真实订阅: 316.sub987.top

**正确方法（V2Board）**:
```javascript
// 1. 取VUE_NAIVE_ACCESS_TOKEN（JSON格式）
raw = localStorage.getItem('VUE_NAIVE_ACCESS_TOKEN')
parsed = JSON.parse(raw)
tok = parsed.value
// 2. 调getSubscribe API
fetch('/api/v1/user/getSubscribe', {headers: {Authorization: tok}})
// 3. 从返回的subscribe_url取真实链接
```

**SSPANEL方法**: 登录→/#/user_info→订阅密钥→`{域名}/link/{token}?sub=1`

### playwright-cli eval语法限制
- 只能用逗号操作符串联：`(a=1, b=2, a+b)`
- 不能用：var / function / for / if / || / ?:
- 可以用的：箭头函数、模板字符串、展开运算符
